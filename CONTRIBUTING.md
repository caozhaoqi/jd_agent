# Contributing（贡献）

> GitLab Community Edition is open source and GitLab Enterprise Edition is open-core.

## 流程

- 1.在以下网址注册 Gitee 账号

```c
https://gitee.com/
```

- 2.注册完成后，找到项目 git 路径使用以下命令拉取代码

```sh
git clone https://gitee.com/caozhaoqi/audio_record_server
```

- 3.下载代码后，使用对应(`README.md`文档)版本工具打开并修改。
- 4.在推送代码前，首先同步云端代码：

```sh
git pull origin master
```

- 5.添加修改文件至暂存区

```sh
git add .
```

- 6.提交代码

```sh
git commit -m 'bug fixed'
```

- 7.推送更新

```sh
git push origin master
```
